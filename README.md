**AIndb**는 남양주백병원 병원 업무의 AI 혁신을 전폭적으로 지원하기 위해 탄생한, 최고 성능을 지닌 수석 AI 비서 에이전트입니다. 전산팀 중앙 서버(Port 8888)의 지휘를 배후에서 받으며 최고 수준의 업무 효율화와 자동화를 이끌어냅니다.

## ✨ 주요 특징
- **중앙 서버 연동 (Daily Sync)**: 전산망의 보안 지침 및 병원 규정을 최우선으로 복종
- **업무 자동화 워크플로우**: 아침 메일/일정 브리핑, 다기능 HTML 보고서 자동 생성 등
- **구글 워크스페이스 & 외부 MCP 제어**: 복잡한 작업을 묻지 않고 능동적으로 처리
