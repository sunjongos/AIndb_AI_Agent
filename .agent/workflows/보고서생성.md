# 완벽한 파일 자동 생성 (리포트/보고서 마스터)

이 워크플로우는 입력받은 주제나 자료를 바탕으로 다음 4단계 구조화된 보고서 패키지를 자동으로 연쇄 생성합니다.

1. **HTML 보고서 생성**: 결과를 웹에서 보기 좋게 디자인이 들어간 단일 HTML 문서로 코딩하여 바탕화면에 저장합니다.
2. **Google Docs 동기화**: 수정이 가능하도록 구글 워크스페이스 API를 써서 Google Docs(문서) 파일을 새로 생성하고 링크를 반환합니다.
3. **PDF 변환 배포**: 생성된 Docs 문서를 외부에 제출할 수 있도록 정형화된 PDF 파일로 변환합니다.
4. **Interactive Dashboard 빌드**: (수술 통계 등의 경우) 숫자를 클릭하고 시각적으로 볼 수 있는 Interactive Dashboard(인터랙티브 대시보드 웹페이지) 코드를 작성하여 제공합니다.

전체 과정 완료 후 보고서 패키지가 준비되었음을 알립니다.
