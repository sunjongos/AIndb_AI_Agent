# 매일 아침 브리핑 (Morning Briefing)

이 워크플로우는 원장님의 원활한 하루 시작을 위해 중요 정보를 수집하고 요약하여 보고합니다.

1. **이메일 요약**: 오늘 도착한 구글 메일(Gmail) 중 중요한 메일 3가지를 가져와 5줄 이내로 핵심만 요약합니다. (구글 워크스페이스 MCP 활용)
2. **일정 브리핑**: 구글 캘린더(Google Calendar)에서 오늘의 수술 일정 및 병원 일정을 추출하여 목록화합니다.
3. **텔레그램 발송**: 요약된 메일 정보와 오늘 일정을 깔끔한 포맷으로 구성하여 원장님의 텔레그램 봇으로 발송합니다.
4. **아바타 활성화**: 출력을 마칠 때 반드시 `![AIndb 비서](aindb_avatar.png)` 마크다운과 함께 쾌활한 인사말을 덧붙입니다.
